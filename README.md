# 📊 Recomendación de Carteras de Inversión con Big Data

## 📌 Descripción del proyecto
Este repositorio contiene el desarrollo de un **proyecto académico** realizado por estudiantes de **tercero de la Universidad Europea** para la asignatura **Proyecto Big Data II**.

El objetivo principal del proyecto es el **diseño e implementación de un sistema de recomendación de carteras de inversión**, basado en el análisis de **datos históricos de activos financieros**, utilizando técnicas de Big Data, análisis estadístico y modelos cuantitativos para apoyar la toma de decisiones de inversión.

---

## 🎯 Objetivos
- Analizar datos históricos de activos financieros.
- Estimar métricas clave como **retorno esperado**, **volatilidad** y **riesgo**.
- Aplicar modelos cuantitativos para la **optimización de carteras**.
- Desarrollar un sistema que recomiende carteras según distintos perfiles de riesgo.
- Evaluar el rendimiento de las carteras propuestas.

---

## 🧠 Metodología
El proyecto sigue las siguientes fases:

1. **Obtención de datos**  
   Recopilación de datos históricos de precios de activos financieros.

2. **Preprocesamiento y EDA**  
   Limpieza de datos, análisis exploratorio y tratamiento de valores nulos.

3. **Cálculo de métricas financieras**  
   - Retornos
   - Volatilidad
   - Correlaciones entre activos

4. **Modelado y optimización**  
   - Modelos de estimación de retornos
   - Optimización de carteras riesgo–retorno

5. **Recomendación de carteras**  
   Generación de carteras adaptadas a distintos perfiles de inversión.
